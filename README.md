# Smart Home Automation using ESP32 and Blynk

## Overview
This project controls home appliances remotely using the ESP32 microcontroller and the Blynk IoT platform.

## Features
- Remote appliance control
- Wi-Fi connectivity
- Real-time switching
- Low-cost IoT solution

## Hardware
- ESP32
- Relay Module
- LED Bulb/DC Motor
- Breadboard
- Jumper Wires

## Software
- Arduino IDE
- Blynk IoT
